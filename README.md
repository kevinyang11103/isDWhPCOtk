# 个性化旅游线路推荐系统 python447

## 项目概述

个性化旅游线路推荐系统致力于解决传统旅游线路的模板化问题，利用数字化与个性化推荐技术，为用户提供定制化的旅游服务。系统简化了旅游信息的管理的流程，通过用户兴趣、时间和预算等数据的分析，实现了智能化的旅游线路推荐。

## 技术栈

### 核心技术

- **开发语言**：Python
- **框架**：Django
- **前端技术**：Vue.js
- **数据库**：MySQL
- **架构**：B/S
- **推荐算法**：协同过滤算法

### 开发工具

- PyCharm
- Navicat

## 功能模块

### 用户角色

- 普通用户
- 管理员

### 普通用户

- **基础操作**
  - 注册
  - 登录
  - 修改个人信息
- **核心功能**
  - 浏览/查询旅游景点、酒店信息、旅游线路
  - 预订酒店与旅游线路，查看订单状态
  - 收藏景点/线路
  - 对景点、酒店、线路进行评论评分

### 管理员

- **用户管理**
  - 增删改查用户账号信息
- **内容管理**
  - 管理景点、酒店、路线信息
  - 审核用户评论
  - 管理旅游新闻
- **订单与系统管理**
  - 查看预订订单
  - 维护系统配置
